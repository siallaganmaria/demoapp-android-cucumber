## Overview

This project is an automation testing framework built using **Cucumber Java**, **JUnit**, **Selenium**, and **Appium**.

The framework is used to automate Android application testing using BDD scenarios written in Gherkin syntax.

---

## Tech Stack

- Java JDK 21
- Maven
- Cucumber Java
- JUnit 4
- Selenium WebDriver
- Appium Java Client

 ## How to Run Test
 Open the runner file:src/test/java/runners/TestRunner.java
 To run a specific feature or scenario, update the tags value inside TestRunner.java
 Example = @login, @buying, @sort

 ## Notes:
 
Android Studio is installed
Android SDK is configured
Emulator is already started
ADB can detect the emulator
